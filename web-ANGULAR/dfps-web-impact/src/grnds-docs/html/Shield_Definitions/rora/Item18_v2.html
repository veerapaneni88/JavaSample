Risk of Recidivism Assessment Definition

18. Alleged perpetrator includes friend/neighbor and/or caretaker


  • No: The current investigation was for self-neglect or did not include a caretaker, neighbor, or friend as an alleged perpetrator.
  • Yes: The current investigation includes a caretaker, neighbor, or friend as an alleged perpetrator.


  • Friend/neighbor: The current investigation involves a perpetrator who is a friend or neighbor with an ongoing relationship with the client.
  • Caretaker: The current investigation involves a primary support person as an alleged perpetrator.
