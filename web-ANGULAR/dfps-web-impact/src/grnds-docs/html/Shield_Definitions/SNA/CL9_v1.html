Strengths and Needs Assessment Definition

CL9. Communication and Literacy (Includes speaking, reading, and writing.)


  1. Able to communicate.
    • Client has no known communication or literacy barriers.
    • Client has some known communication or literacy barriers but is able to function independently.

  2. Able to communicate with minimal assistance.
    • Client has minimal known communication or literacy barriers (e.g., language differences, able to communicate verbally but cannot write, etc.); AND
    • Client does not have the information and/or resources needed to access supportive services/adaptive devices (e.g., an interpreter).

  3. Significant communication or literacy barriers.
    • Client has significant known communication or literacy barriers (e.g., language differences for which translator services are not readily available, client is not able to communicate verbally or through writing, etc.) AND
    • Client does not have the ability or means to access supportive services/adaptive devices (e.g., an interpreter).
